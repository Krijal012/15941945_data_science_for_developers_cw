# Data Science Coursework - Norfolk vs Suffolk

## Structure
- `Clean Data/` - Final cleaned datasets used for analysis
- `R Codes/` - All R scripts (cleaning, EDA, modeling, recommendation)
- `Charts (Visualizations)/` - Exported EDA and model visualizations
- `Report.pdf` - Final written report

## How to run
1. Open the `.Rproj` file in RStudio
2. Run scripts in `R Codes/` in numbered order
3. Cleaned data is read from `Clean Data/`, charts saved to `Charts (Visualizations)/`
